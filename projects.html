Machine Learning Projects

Piazza Classifier

Tools: C++ | Machine Learning | Docker

To classify student posts by subject, I used natural language processing and machine learning techniques in C++ to classify student forum posts.

This model calculates log-probability scores for student posts to assign subject labels and achieves a 87% accuracy rate!

Aviator Recognition

Tools: Python | Machine Learning | Arduino | Circuitry

I used Python and Tensorflow to build machine learning model that performs classification through convolutional neural networks in order to distinguish birds from a user-given photo.

Furthermore, I built an Arduino robotic prototype that detects aviators traveling too closely to solar panels with an ultrasonic sensor and responds with a patterned strobing light.
